## Task 4: Create a GET /todos/:id Endpoint

**Description:**  
Create a GET endpoint `/todos/:id` that responds with the todo with the given id, or 404 if not found.

**Example:**
```js
// GET /todos/1 => { id: 1, title: 'Buy milk', completed: false }
```
