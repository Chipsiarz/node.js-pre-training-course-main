## Task 1: Create a GET /todos Endpoint

**Description:**  
Create an Express.js server with a GET endpoint `/todos` that responds with a list of ToDo items (array of objects with id, title, completed).

**Example:**
```js
// GET /todos
// Response: [ { id: 1, title: 'Buy milk', completed: false } ]
```
