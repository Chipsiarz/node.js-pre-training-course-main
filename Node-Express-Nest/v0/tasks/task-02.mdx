## Task 2: Create a POST /todos Endpoint

**Description:**  
Add a POST endpoint `/todos` that accepts a todo object in the request body and adds it to the list. Respond with the created todo.

**Example:**
```js
// POST /todos { "title": "Read book" }
// Response: { id: 2, title: 'Read book', completed: false }
```
